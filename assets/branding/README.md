# Branding Assets

Replace the placeholder SVG files in this folder with your actual logo files.

## Files

| File | Purpose | Recommended Size | Used In |
|------|---------|-----------------|---------|
| `logo.svg` | Main company logo | ~300x80px (or similar aspect ratio) | Header, Login page |
| `favicon.svg` | Browser tab icon | 32x32px or 100x100px square | Browser tab |
| `hospital-logo.svg` | Hospital/client logo | ~250x60px (or similar aspect ratio) | Login page (below main logo) |

## Notes

- You can use `.svg`, `.png`, or `.jpg` files — just update the file references in `index.html` if you change the extension.
- For the favicon, `.svg`, `.ico`, or `.png` all work in modern browsers.
- Keep logo files reasonably small (under 100KB) for fast page loads.
